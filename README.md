使用jsDelivr加速GitHub图片  </br>
本地使用[PicGo](https://github.com/PicGo)上传图片
## e.g
>仓库名:    zhboat/cdn_pic  <br/>
 分支名:    master  </br>
 token:   [github添加token](https://github.com/settings/tokens)  </br>
 存储路径:  不设置即根目录  </br>
 自定义域名:使用https://cdn.jsdelivr.net/gh/urname/reponame/  </br>
